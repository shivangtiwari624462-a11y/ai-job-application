# AI Job Application Automation System

## Overview
This is a Python-based automation tool built using Selenium to automate job search and application workflows for learning purposes.

## Features
- Automated job search
- Easy Apply detection
- Form auto-filling automation
- Resume optimization logic
- Email reporting system
- Modular structure for multiple platforms

## Tech Stack
- Python
- Selenium
- Chromedriver
- Environment Variables
- Automation Logic

## Project Purpose
This project was developed as a learning-based automation system to understand browser automation, workflow optimization, and AI integration. It is built for educational and experimental purposes.

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Configure environment variables (.env file)

3. Run the project:
   python main.py
